## 動作確認のお願い

LaTeXの作成環境によって，利用できるコンパイラ，パッケージ，フォント，設定画面などには多少の違いがあります．  
ただし，使用するコンパイル方式や必要なファイルを揃えれば，基本的にはどの環境でも同等の出力結果を得られます．

本リポジトリでは，VS Code，Overleaf，CloudLaTeXの各環境に対応したファイルを用意しています．  
コンパイル方式や環境ごとの設定について詳しく知りたい場合は，`LaTeX_explanation_Overleaf.pdf`を参照してください．

各自で次のいずれかの環境を確認し，正常にコンパイルできるか確認してください．

## 手順

1. GitHubからこのリポジトリをZIP形式でダウンロードする．

2. 担当する環境に応じて，プロジェクトを開く．

   - **VS Code（TeX Live）**
     - ZIPファイルを展開する．
     - 展開したフォルダをVS Codeで開く．

   - **Overleaf**
     - `New Project` → `Upload Project`を選択する．
     - ダウンロードしたZIPファイルをアップロードする．

   - **CloudLaTeX**
     - 新しいプロジェクトを作成する．
     - ZIPファイル，または展開したプロジェクト内のファイルをアップロードする．

3. 担当する環境に対応したファイルを開く．

   | 使用環境 | 使用するファイル |
   |---|---|
   | VS Code（TeX Live） | `LaTeX_practice_VSCode.tex` |
   | Overleaf | `LaTeX_practice_Overleaf.tex` |
   | CloudLaTeX | `LaTeX_practice_CloudLaTeX.tex` |

4. コンパイラを設定する．

   本確認では，すべての環境で同じコンパイル方式を使用する．  
   各環境の設定画面やビルド設定から，**LuaLaTeX**を選択すること．

   - VS Codeでは，LaTeX Workshopなどのビルド設定でLuaLaTeXを選択する．
   - Overleafでは，`File` → `Settings` → `Compiler`から`LaTeX`を選択する．さらに，`latexmkrc`を必ず入れる．
   - CloudLaTeXでは，プロジェクトのコンパイラ設定からLuaLaTeXを選択する．

5. 対応する練習用ファイルをコンパイルする．

6. 同じ環境で，対応する解答用ファイルもコンパイルする．

   解答用ファイルについても，ファイル名の`practice`に続いて`answer`が付いたものを使用する．

   例：

   - `LaTeX_practice_VSCode.tex`
   - `LaTeX_practice_answer_VSCode.tex`

7. 練習用ファイルと解答用ファイルの両方について，以下の項目を確認する．

## 確認してほしいこと

- PDFが最後のページまで生成されるか
- 日本語が正しく表示されるか
- 画像が正しく表示されるか
- 表の幅や配置が崩れていないか
- 箇条書きが正しく表示されるか
- ソースコードが枠内に正しく表示されるか
- 参考文献が本文から引用され，文書末尾に表示されるか
- 二段組の図がページ全体の幅で正しく表示されるか
- エラーや重大な警告が発生していないか
